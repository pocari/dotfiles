#Setup

##win32

- clone from github

  ```
  cd %USERPROFILE%
  git clone  git@github.com:pocari/dotfiles.git
  ```
  
- setup links(vim setting only)

  ```
  run_setup.bat
  ```

## macos, linux
- clone from github
  
  ```
  cd ~/
  git clone git@github.com:pocari/dotfiles.git
  ```
  
- setup links

  ```
  sh setup.sh
  ```
